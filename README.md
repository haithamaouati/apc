## Installation

1. Update & upgrade packages
    ```
    pkg update -yy && apt upgrade -yy
    ```
    
2. Install git package
    ```
    pkg install git -yy
    ```

3. Clone the repository
    ```
    git clone https://github.com/haithamaouati/apc.git
    ```
4. Change to apc directory
    ```
    cd apc
    ```
    
5. Change the access mode
    ```
    chmod +x apc.sh
    ```

## Usage

6. Run the script
